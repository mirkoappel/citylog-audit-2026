# Memory Index

Persistente Wissensbasis des Forschungsagenten. Wird zu Beginn jedes neuen Kontextfensters gelesen.

- [Projekt](memory/project.md) — Projektziel, harte Regeln, aktueller Forschungsstand, nächste Schritte
- [Erkenntnisse](memory/erkenntnisse.md) — Eigene Experiment-Ergebnisse, Befunde, Selbstreflexion, offene Hypothesen
- [Synthese](memory/synthese.md) — Übergeordnete Muster, Vorhersagen, freie Synthese — entsteht im Dreaming-Modus
- [Hypothesen](memory/hypothesen.md) — Offene Forschungsfragen, priorisierte nächste Experimente, bestätigte und widerlegte Hypothesen
- [Feedback](memory/feedback.md) — Erwartungen des Nutzers, Arbeitsweise, was zu vermeiden ist
- [Changelog](memory/changelog.md) — Evolutionslog des Systems — strukturelle Entscheidungen, Anpassungen, Begründungen
- [Zustand](memory/zustand.md) — Aktueller Forschungskompass — welches Signal ist gerade aktiv und was legt es nahe
- [Notizen](memory/notizen.md) — Scratchpad für aktuelle komplexe Aufgaben — Zwischengedanken, Teilschritte, halbfertige Überlegungen
- [Bibliothek](recherche/README.md) — Übersicht über alle Recherche-Dokumente in `recherche/`
